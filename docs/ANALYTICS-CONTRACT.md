# PocketPills Analytics Data Contract

The canonical analytics contract is now hosted in pp-docs and is browsable per section:

**https://pp-docs.pages.dev/guides/v3**

The pp-docs site mirrors what used to live here, split into navigable guides (architecture, identity model, session model, event lifecycle, attribution, cookie contract, persistence scenarios, debugging, reference). Cross-references between sections are proper internal links, and the search index (Pagefind) covers every section. Concept-level pages live under `/concepts/v3/`; API references under `/api/v3/`.

To update the analytics contract, edit the MDoc files in [`pp-docs/content/guides/v3/`](https://github.com/BishalSahoo-Pocketpills/pp-docs/tree/main/content/guides/v3) (or `content/concepts/v3/` / `content/api/v3/`) and open a PR there. The site auto-deploys to Cloudflare Pages on merge.
